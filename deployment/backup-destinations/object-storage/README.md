# Object Storage

Backup destination is a storage location where vProtect keeps VMs, Containers, Cloud and applications backup copies. vProtect supports different type of object storage.

* [AWS S3](aws-s3-or-s3-compatible.md)
* [Google Cloud Storage](google-cloud-storage.md)
* [Microsoft Azure Blob Storage](microsoft-azure-blob-storage.md)
* [OpenStack SWIFT](openstack-swift.md)
* [IBM Cloud Object Storage](ibm-cloud-object-storage.md)
* [Oracle Cloud Infrastructure Object Storage](oracle-cloud-infrastructure-object-storage.md)
* [Scality RING](scality-ring.md)
* [Ceph Rados Gateway](ceph-rados-gateway.md)
